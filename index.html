Servicio de Verificación de Identidad con IA

Estado del servicio: ONLINE ✅

Instrucciones de la Petición (Request)

Realiza una petición HTTP de tipo POST hacia el siguiente endpoint:

Ruta: /imagenes/

Parámetros requeridos (form-data):

  • fotoDni: Imagen correspondiente al documento de identidad.
  • imgSlf: Fotografía tipo "selfie" del usuario.

Pruebas Rápidas (Interactive Test)

¿Quieres probar la API directamente desde tu navegador sin usar código ni Postman?

FastAPI autogenera una documentación interactiva (Swagger UI) donde puedes cargar tus imágenes y ejecutar el endpoint en tiempo real.

Ir al panel de pruebas (/docs)

Estructura de la Respuesta (Response)

🟢 Si el proceso es correcto (HTTP 200):

El servidor no responderá con texto ni JSON. Devolverá directamente un flujo de datos binarios correspondiente a la imagen resultante optimizada.

Cabecera devuelta: Content-Type: image/webp

🔴 Si ocurre un error (HTTP 400 / 500):

El servidor devolverá un objeto JSON indicando el fallo del procesamiento:

{
  "error": "No se pudo codificar la imagen"
}
